# Proyecto Redes Avanzadas en Gns3

Automatización de laboratorio en GNS3 con Docker Python, Cisco IOSv y MikroTik

## Equipos
- R1: Cisco IOSv
- R2: Cisco IOSv
- R3: MikroTik CHR
- Docker Python: nodo de automatización

## Funciones
- Automatización Telnet/SSH en R1 y R2
- Consumo API REST en R3
- Configuración de rutas estáticas
- Configuración de IPsec entre R1 y R3

## Archivos
- config.py
- automate_r2.py
- automate_r1.py
- automate_r3.py
- main.py

## Dependencias
- netmiko
- requests
